# JavaScript Functions : String manipulation, Array and Mathematical.

A clean, well-commented collection of essential functions for **String Manipulation**, **Array Operations**, and **Mathematical Calculations** — written in JavaScript.

## Features

### String Manipulation
- `reverseString(str)` — reverse without using `.reverse()`
- `countCharacters(str)` — manual character count
- `capitalizeWords(sentence)` — title case (first letter of each word)

### Array Functions
- `findMaximum(arr)` / `findMinimum(arr)`
- `sumOfArray(arr)`
- `filterArray(arr, condition)` — e.g. `x => x > 0`

### Mathematical Functions
- `factorial(n)` — iterative
- `isPrime(n)` — optimized O(√n) check
- `fibonacciSequence(terms)` — returns array `[0, 1, 1, 2, 3, ...]`
